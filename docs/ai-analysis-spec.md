## 1. 抖音指数分析

### 数据来源
- **订阅页**（`creator.douyin.com`）：搜索指数、综合指数、日环比
- **关键词页**（`creator.douyin.com/arithmetic-index`）：关联词TOP10、人群画像

### 核心指标

| 指标 | 定义 | 说明 |
|------|------|------|
| **搜索指数** | 用户主动搜索行为的量化 | 高 = 潜在游客主动在找这个景区 |
| **综合指数** | 内容热度+互动热度+搜索热度的综合 | 平台加权计算，权重不公开 |
| **日环比** | 较前一日的变化百分比 | ±5%以内为正常波动，±20%以上标记🔺异动 |

### 分析逻辑
1. 读取订阅页8个景区的搜索指数和综合指数
2. 计算日环比，标记异常波动（阈值：±20%）
3. 对比各景区相对位次变化
4. 结合竞品动态，归因异动原因（如：节日/竞品动作/爆款内容）
5. 输出简报：指数+环比+位次+归因

### ⚠️ 重要说明
> **综合指数的权重不公开**。抖音官方未公布各维度权重，任何声称知道权重比例（如"搜索*0.4+内容*0.3"）的公式均为推测，不可靠。
>
> 我们采用**双指标交叉验证**：搜索指数（用户主动意图）和综合指数（平台推荐流量）共同参考，相互印证，单一指标不作为结论依据。

## 2. 小红书灵犀数据分析

### 数据来源
- **灵犀后台**（`idea.xiaohongshu.com`）：搜索总量、人群资产、行业排名
- **探索页**（`xiaohongshu.com/explore`）：爆款笔记标题、点赞、收藏

### 核心指标

| 指标 | 定义 | 说明 |
|------|------|------|
| **搜索总量** | 关键词在统计周期内的搜索次数 | 反映主动需求规模 |
| **人群资产** | 近30天与该词互动的人群去重总量 | 反映内容覆盖人群规模 |
| **行业搜索排名** | 该词在所属行业中的搜索量位次 | 排名越高表示需求越集中 |
| **互动总量** | 笔记点赞+收藏+评论之和 | 反映内容质量 |

### 分析逻辑
1. 灵犀后台抓取搜索总量和人群资产
2. 探索页抓取相关爆款笔记（点赞>1000）
3. 识别内容规律：标题公式/标签/博主类型
4. 输出：搜索量级+人群覆盖+爆款内容特征

## 3. 竞品爆款拆解

### 触发条件
- 搜索指数日环比 > +50%
- 或小红书单篇笔记点赞 > 5万
- 或全平台声量明显上涨

### 分析维度

| 维度 | 数据来源 | 说明 |
|------|---------|------|
| 抖音爆款 | 关键词页关联分析 | 提取高赞视频标题/内容特征 |
| 小红书爆款 | 探索页截图+ARIA快照 | 提取爆款笔记标签/博主类型 |
| 微博爆款 | 搜索结果 | 转发>1000的帖子 |
| 百度舆情 | 百度搜索结果 | 搜索词联想和负面监测 |

### 归因分析框架

```
数据异动（如搜索指数+70%）
    ↓
溯源：哪个平台/哪篇内容/哪个时间点引发
    ↓
归因：情绪共鸣？稀缺体验？达人带动？节庆效应？
    ↓
行动建议：内容公式借鉴 / 借势营销 / 优化方向
```

## 4. 周度客流洞察分析

### 数据来源
- 每日CSV客流数据（桌面同步）
- 《穿越德化街》演出数据（Excel）
- 历史对比（2023-2025年同期）

### 核心指标

| 指标 | 计算方式 | 说明 |
|------|---------|------|
| **转化率** | 观演人次 ÷ 入园人次 | 核心效率指标 |
| **上座率** | 观演人次 ÷ 座位总量 | 场次利用效率 |
| **散客占比** | 散客人次 ÷ 总人次 | 渠道结构 |
| **同比/环比** | vs上年同期/vs上月 | 趋势判断 |

### 分析逻辑
1. 读取周度CSV数据，计算各项指标
2. 对比历史同期（W16/W17/W52）
3. 散客/渠道拆分分析
4. 《穿越德化街》专项分析（场次/上座率/场次结构）
5. 输出：数据+趋势+归因+建议

## 5. 知识层 INGEST 逻辑

### 流程
```
每周日晚自动触发
    ↓
扫描本周新增内容（竞品动态/案例库/日报）
    ↓
LLM 提炼：数据异动 + 归因 + 行动建议
    ↓
更新到 concepts/ 或 entities/
    ↓
生成 lint 检查报告
```

### LLM 提炼标准
- 每条洞察不超过3句话
- 必须包含：事实 + 意义 + 归属目录
- 禁止：无依据猜测 / 重复已有结论 / 与现有关键知识冲突
