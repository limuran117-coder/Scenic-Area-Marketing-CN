## 工具可用性说明（2026-04-30）

1. **小红书搜索（xiaohongshu-crawler）**：Cookie 有效但页面被风控拦截，搜索结果为空
2. **web_search**：需配置 MiniMax API key（MINIMAX_CODE_PLAN_KEY / MINIMAX_API_KEY）
3. **飞书/B站/抖音等平台**：均需浏览器自动化或登录态，当前环境不可用

**建议**：配置 MiniMax API key 或安装 Playwright 浏览器后重试
