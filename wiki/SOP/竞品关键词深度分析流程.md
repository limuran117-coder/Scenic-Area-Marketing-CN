## 一、核心原则

每次分析必须同时覆盖以下**四个平台**，缺一不可：
1. 抖音指数（关键词+关联词+人群画像）
2. 小红书灵犀（搜索洞察）
3. 小红书搜索结果页（爆款内容）
4. 百度搜索（运营/财务/媒体报道）

---

## 二、四平台操作流程

### 平台1：抖音指数（核心平台）

**目标URL：** https://creator.douyin.com/creator-micro/creator-count/arithmetic-index

**CDP端口：** 18800（专属浏览器）

**执行步骤：**

**Step 1：用cdp_keyword_deep.py脚本采集（推荐）**
```bash
python3 ~/.openclaw/workspace/scripts/cdp_keyword_deep.py "关键词"
```
脚本会自动完成：搜索框输入→滚轮下滑→关联分析tab→人群分析tab

**必须采集的三个维度：**
- 搜索指数 + 综合指数（数值+日环比）
- 关联词TOP10（搜索上游词，反映用户决策路径）
- **人群画像（完整4字段：地域分布TOP10含TGI + 性别比例 + 年龄段分布 + TGI指数）⚠️不得缺失**

**Step 2：验证数据完整性**
检查返回数据是否包含：
- `related_words` 数组（至少5条）
- `crowd_profile.地域分布` 数组（至少10条）
- `crowd_profile.性别` 字段

---

### 平台2：小红书灵犀

**目标URL：** https://idea.xiaohongshu.com/idea/trend/trendAnalyze

**操作方式：** 打开专属浏览器Tab4（小红书灵犀）

**执行步骤：**
1. 在关键词搜索框输入目标关键词
2. 使用DOM注入脚本触发搜索（React组件普通type无效）：
```javascript
const inp = document.querySelector('input.d-text');
if (inp) { inp.focus(); inp.value = '关键词'; inp.dispatchEvent(new Event('input', {bubbles:true})); }
```
3. 等2秒后点击下拉选项
4. 等待数据加载，采集：
   - 搜索总量（万为单位）
   - 环比/同比变化
   - 热门搜索词TOP10
   - 搜索上下游词

**采集字段清单：**
| 字段 | 说明 | 示例 |
|------|------|------|
| 搜索总量 | 近7天搜索次数 | 12.5万 |
| 环比 | 较上周期变化 | +16.72% |
| 同比 | 较去年同期变化 | -8.70% |
| 热门搜索词 | 搜索量降序TOP10 | 银基动物王国攻略/门票/奇翼飞鹰等 |
| 上游词 | 用户搜索前的行为 | 去银基需要准备什么 |
| 下游词 | 用户搜索后的行为 | 银基住哪里好 |

---

### 平台3：小红书搜索结果页

**目标URL：** https://www.xiaohongshu.com/search_result?keyword={关键词}

**操作方式：** 打开新Tab直接导航到URL（禁止操作Explore页）

**采集内容：**
- 爆款笔记TOP10（点赞/收藏/评论）
- 筛选标签分布
- 今日热搜问题
- 博主类型（素人/达人/KOC比例）

**⚠️ 禁止事项：**
- 禁止点击Explore页（自动播放视频会锁死DOM）
- 禁止使用搜索框内的提示词，必须用固定关键词直接搜

---

### 平台4：百度搜索

**目标URL：** https://www.baidu.com/s?wd={关键词}&rn=20

**操作方式：** 打开新Tab直接导航到URL（每次开新Tab防验证）

**采集内容：**
| 类型 | 采集字段 |
|------|---------|
| 基础信息 | 地址/票价/开放时间/规模 |
| 营收数据 | 最新营收/年客流量/人次均价 |
| 媒体报道 | 正面/负面/中性文章标题 |
| 热搜问题 | 相关搜索TOP10 |

## 三、报告标准格式（7大必须模块）

每次分析报告必须包含以下全部章节：

### 模块1：核心数据快照表
四平台关键指标汇总，涨跌用emoji标注

### 模块2：抖音指数分析
- 搜索/综合指数+日环比
- **⚠️ 人群画像（必须完整采集）：**
  - 地域分布TOP10（含TGI指数）
  - 性别比例（男/女）
  - 年龄段分布（18-23/24-30/31-40/41-50/50+）
  - 兴趣偏好标签（如有）
- 核心洞察

**⚠️ 验证要求：人群画像4个字段（地域/性别/年龄段/TGI）必须全部有数据，空缺字段需注明原因并用"—"标注，不得留空**

### 模块3：小红书灵犀数据
- 搜索总量+环比/同比
- 热门搜索词TOP5
- 决策路径分析

### 模块4：小红书内容分析
- 爆款TOP3特点描述
- 热搜问题
- 内容营销规律

### 模块5：百度数据
- 票价/规模/运营信息
- 媒体报道摘要

### 模块6：内容营销规律总结
- 爆款公式提炼
- 用户决策路径
- 季节/节假日规律

### 模块7：对电影小镇的深度借鉴
至少3条可执行的具体建议

## 六、常见问题

**Q：关联词数据为空？**
A：可能是页面未完全加载，等待3秒后重试

**Q：抖音指数2026年5月10日01:00-07:00停服？**
A：留意维护窗口，跳过该时段采集

**Q：CDP连接失败？**
A：检查端口是否为18800（专属浏览器），不是9222

**Q：小红书灵犀输入框无反应？**
A：用DOM注入脚本，而非直接type()
