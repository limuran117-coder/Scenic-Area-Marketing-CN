## douyin_index_v9.py

**位置：** `~/.openclaw/workspace/scripts/douyin_index_v9.py`

**功能：** 抖音创作者后台订阅数据采集（8个竞品景区）

**采集URL：** `https://creator.douyin.com/creator-micro/creator-count/my-subscript?source=creator`

**采集逻辑：**
1. 连接专属浏览器 CDP (18800)
2. 导航到订阅页
3. 检测页面数据日期，若非今日 → `page.reload()` 等待8秒（**v2026-04-22修复**）
4. 读取页面文本，正则解析景区名+搜索指数+综合指数
5. 输出JSON到 `/tmp/crawl_data_douyin_YYYYMMDD.json`
6. 关闭Page，保留Browser

**输出格式：**
```json
{
  "date": "2026-04-22",
  "crawled_at": "2026-04-22T10:30:00",
  "competitors": [
    {"name": "万岁山武侠城", "search": 47390, "synth": 24952},
    ...
  ]
}
```

**输出文件：** `/tmp/crawl_data_douyin_YYYYMMDD.json`

## send_feishu_card.py

**位置：** `~/.openclaw/workspace/scripts/send_feishu_card.py`

**功能：** 飞书卡片消息发送

**用法：**
```bash
python3 send_feishu_card.py <chat_id> <card_json>
```

**发送目标：**
| 群名 | chat_id |
|------|---------|
| 电影小镇群 | oc_2581c03b79e4893cc3616b253d60f34e |
| 数据推送群 | oc_f109bcfd1bc7e166fd0ae077f70247cf |

**卡片格式规范（v2026-04-19确认）：**
- `header.title.tag = "plain_text"` + `header.title.content`
- 内容放 `body.elements[]`
- 每个表格完整放在一个markdown元素内
- 每卡 ≤ 5个表格

**私信发送：** chat_id改为用户open_id（send_feishu_card.py私信版）

## 浏览器连接策略

所有脚本的CDP连接顺序：

```
1. 尝试连接 http://127.0.0.1:18800 （专属浏览器）
2. 若失败，尝试 http://127.0.0.1:9222 （站长Chrome）
3. 若均失败，启动临时Chrome
```

**专属浏览器端口：** 18800
**专属浏览器Profile：** openclaw
**Cookie存储：** 独立于站长Chrome
