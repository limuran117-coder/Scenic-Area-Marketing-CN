## 卡片格式规范

### 正确格式（v2026-04-19确认）⚠️ 铁律

```python
card = {
    "schema": "2.0",
    "header": {
        "title": {
            "tag": "plain_text",
            "content": "标题内容"
        }
    },
    "body": {
        "elements": [
            {"tag": "markdown", "content": "第一段文字或完整表格"},
            {"tag": "markdown", "content": "第二段..."}
        ]
    }
}
```

### 错误格式（历史上犯过的）

❌ `header.title = "字符串"`（少了tag+content嵌套）
❌ `body.elements[]`（少了body包装）
❌ 表格拆成多个markdown元素（表头和行分开）
❌ schema用"1.0"或省略

### 发送脚本

```python
from send_feishu_card import send_card
result = send_card("oc_群id", card)
```

## 投递配置说明

| 模式 | 含义 |
|------|------|
| `announce` | cron执行完成后，Gateway自动发送完成通知 |
| `announce->last` | 发送到last_active_session，有飞书路由才生效 |
| `none` | 不发送announce，只靠Python脚本内部发卡 |

**当前策略：** Python脚本负责发卡（可靠），外层announce一律`none`。
