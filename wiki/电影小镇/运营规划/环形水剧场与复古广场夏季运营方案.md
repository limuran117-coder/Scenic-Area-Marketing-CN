# 环形水剧场 & 复古广场夏季运营方案

## 一、环形水剧场（2000座室外）

### 1.1 场地资产禀赋

| 资产项 | 现状 |
|--------|------|
| 容量 | 2000座 |
| 形态 | 圆形露天剧场，环形布局 |
| 水体 | 场地内有水面条件 |
| 夜间演出 | 仅打铁花 |
| 白天状态 | 空置 |

**全国中等规模圆形剧场参照：**
- 新加坡Night Safari户外剧场（年客流200万+）：轻资产重舞美，验证中等规模圆形剧场完全可支撑驻场演出
- 日本京都鸭川"川床"文化：把自然条件（河边+夜风）变成演出资产，水面+夜空是天然资产
- 曼谷Asiatique河畔夜市：演艺+餐饮+氛围组合拳，商业化运营可参考

**核心判断：** 场地条件优秀（圆形+水面+2000座），但内容严重不足，打铁花是唯一资产，场地利用率极低。

### 1.2 白天方案（14:00-18:30）

**核心原则：不做演出，做高温避暑目的地**

郑州夏季日落时间参考：
| 月份 | 日落时间（郑州） |
|------|----------------|
| 5月上旬 | 约19:10 |
| 6-7月 | 约19:30-19:40 |
| 8月 | 约19:15 |

**14:00-17:00光照判断：** 强日光直射，任何舞美灯光效果都被压制，做演出=浪费内容。正确逻辑是把水剧场变成"游客必须停下来避暑"的刚需节点。

| 时段 | 活动 | 说明 |
|------|------|------|
| 全天 | 雾森常态开放 | 把水剧场变成园区避暑核心点，游客停留=消费可能 |
| 14:00/15:00/16:00 | 泡泡派对（每场20分钟） | 泡泡机+雾森+背景BGM，孩子导向，家长休憩 |
| 17:00-18:00 | 泼水互动节（每20分钟一场） | 演员带队水枪大战，天色仍亮但阳光已斜，泼水视觉清晰，与傍晚无缝衔接 |

**注：** 不做光影预热秀，强光下灯光无效。泡泡派对目标不是培养看演出习惯，而是让人留在水剧场。

### 1.3 夜间方案（19:30-21:00）

**《凤求凰》大型水幕光影秀**

#### 选材依据

| 故事母题 | 情绪出口 | 适合原因 |
|---------|---------|---------|
| 梁祝化蝶 | 悲剧唯美 | 情绪偏悲，游客从欢乐景区看悲剧出戏 |
| 白蛇传 | 正义战胜邪恶 | 有打斗有温情，结局圆满 |
| 牛郎织女 | 爱情团圆 | 七夕节点强，但日常演出缺乏独特性 |
| **凤求凰** | **才子佳人** | **民国风无缝衔接，喜剧结局** |

#### 为什么凤求凰是金矿

- 电影小镇用民国手法解构演绎司马相如与卓文君故事——服装/舞美/音乐全面民国化，与景区调性完全一致
- 结局"才子佳人终成眷属"，情绪正向，适合旅游高峰值体验
- 表演形式：书生抚琴+佳人伴舞+水面镜像+激光化蝶，2-3名主演，舞美承担80%效果

#### 演出结构（40分钟）

| 章节 | 内容 | 视觉元素 |
|------|------|---------|
| 第一章·琴动文君 | 司马相如抚琴，佳人以舞作答 | 水面投影+雾森+古琴BGM |
| 第二章·凤求凰 | 双人舞高潮，月下相遇 | 激光化蝶效果+水面灯光 |
| 第三章·当垆卖酒 | 烟火人间，喜剧呈现 | 烟火+暖色灯光+互动表演 |
| 尾声·白首之约 | 圆满结局，众人祝福 | 全场灯光+水面焰火 |

#### 设备需求

| 类型 | 数量 | 说明 |
|------|------|------|
| 激光投影设备 | 1套 | 核心视觉担当，需专业采购 |
| 水幕发生器 | 2-3组 | 投射画面用 |
| 防水音响 | 若干 | 户外防水规格 |
| 雾森系统 | 已有/需确认 | 现有设备是否支持本场地 |
| 主演 | 2-3人 | 舞蹈演员为主 |
| 群众演员 | 5-8人 | 当垆卖酒段落互动 |

轻资产模式，预计百万级投入。

## 二、复古广场（4000㎡，60×80m）

### 2.1 场地资产禀赋

| 资产项 | 现状 |
|--------|------|
| 面积 | 约4000㎡（60m×80m） |
| 地面 | 平整水泥地 |
| 设备 | 大屏+灯光+音响+摇臂转播 |
| 降温系统 | 喷淋+雾森（全铺或局部待确认） |
| 风格 | 80s/90s复古 |

**核心判断：** 硬件优秀，内容策略核心是"嬉水+音乐"双主线，从傍晚接力至夜间，形成闭环。

### 2.2 核心策略

**定位：复古嬉水音乐广场——中国版户外CLUB**

三层内容逻辑：
1. **嬉水是钩子** — 雾森+喷淋解决"太热不想出门"的核心痛点，把人从室内拉出来
2. **音乐是留客** — 80s/90s音乐+Live驻场让人停留，不是背景音是主角
3. **抖音是放大器** — 把游客体验变成他们主动传播的素材

### 2.3 每日时间轴（含日落校准）

郑州夏季典型日落：19:10-19:40

| 时间 | 天色状态 | 内容 | 说明 |
|------|---------|------|------|
| 17:00-18:30 | 日落前，阳光逐渐柔和 | 嬉水自由时间 | 雾森全开+定时喷淋，背景BGM循环80s金曲 |
| 18:30-19:30 | **日落黄金30分钟** | 复古DJ暖场 | 大屏播放80s MTV片段+DJ打碟；这是全天最精华的30分钟：日落+音乐+雾森三件事同时发生，刻意保护不被任何内容打断 |
| 19:30-20:30 | 天黑不久 | Live乐队驻场 | 本地乐队翻唱/改编80s金曲，气氛推向高潮 |
| 20:30-21:00 | 夜色已深 | 全场泼水压轴 | NPC+演员带队泼水狂欢；刻意设计的抖音爆点 |
| 21:00-22:00 | 深夜 | After Party | DJ收尾，游客自由舞蹈直到清场 |

### 2.4 关键爆点：20:30泼水压轴

**设计逻辑：** 把压轴环节设计成"游客不拍会后悔"的画面。

**国内外参照：**
- 泰国考山路夜市：每晚21:00水战，游客自发参与
- 西班牙Ibiza海滩：日落泼香槟仪式
- 上海外滩音乐广场：喷淋降温+定时音乐表演，站立比坐着更有现场感
- 西安大唐不夜城：不间断轮转音乐+节点式定点大show

**具体设计：** 演员手持洒水工具带领全场泼水，大屏同步播放音乐，雾森全开形成"水雾+灯光"迷幻感。

**为什么有效：** 游客自己拍的视频比官方拍的更有传播力。几千人在广场泼水狂欢——这个画面本身就是内容，不需要任何策划。

## 三、两个场地协同闭环

### 3.1 一日游客动线

```
14:00-18:00  →  环形水剧场（泡泡派对/泼水解暑）
17:00-18:30  →  复古广场（嬉水自由时间，雾森全开）
18:30-19:30  →  复古广场（日落黄金时段🌅，DJ暖场）
19:30-20:30  →  复古广场（Live乐队驻场）
20:30-21:00  →  复古广场泼水压轴（抖音爆点）→ 结束后移步
20:00-21:00  →  环形水剧场（《凤求凰》水幕光影秀）
```

### 3.2 闭环逻辑

**嬉水解暑（环形）→ 音乐留客（复古广场）→ 泼水压轴（复古广场）→ 夜间演出收尾（环形）**

两个场地形成完整闭环，覆盖游客从白天到夜晚的全程体验，动线自然，不走回头路。

## 四、待确认问题

| 优先级 | 问题 | 为什么重要 |
|--------|------|-----------|
| ① | 雾森+喷淋覆盖面积（全铺还是局部） | 全铺/局部直接影响嬉水区范围和路线设计 |
| ② | 音响功率是否支持3000人 | 3000人同时在广场，音量能否覆盖全场而不炸 |
| ③ | 摇臂是否有专人操作 | 有专人→可做互动直播；固定位→只能做静态转播 |
| ④ | 广场地面排水条件 | 泼水后地面湿滑，是否需要特殊地面处理 |
