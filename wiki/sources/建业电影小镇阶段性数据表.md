# 建业电影小镇阶段性数据表

**类型**：一手数据源
**原始文件**：`wiki/raw/建业电影小镇阶段性数据表.xlsx`
**日期**：2026-04-25

## 7个Sheet汇总

| Sheet | 核心内容 |
|-------|---------|
| 基础信息年度 | 2023-2025三年年度核心数据对比 |
| 基础信息1-3月 | 四年Q1数据对比 + 客单价分析结论 |
| 平日 大假数据 | 平日与大假的演出/观演/收入/转化率拆分 |
| 模拟纯德化街收入 | 2023-2026四年德化街专项收入模拟 |
| 2025年激励数据 | 7-8月激励政策效果对比（有/无激励） |
| 指标制定及拆分 | 2026年全年12个月指标拆分（上座率/转化率/场次） |
| 排期模型 | 按客流量分级的演出排期方案（0-2000到15000+） |
