## 二、尺寸规范

### viewBox 宽度选择
- **窄幅（1200）**：单行内容、Step流程 → 图7/8/9/10/11
- **宽幅（1400）**：多列并行、需要并排对比 → 图2/3/4/5/6/12

### viewBox 高度选择
根据内容总高度估算，公式：
```
总高度 = 各区块height之和 + 各区块间隙之和 + 标题(50) + 图例(30) + 边距(40)
```

**经验值：**
- 10个以内内容块 → 总高 800-900
- 15个以内内容块 → 总高 1000-1100
- 20个以上内容块 → 总高 1200+

### 单个区块高度参考
| 区块类型 | 高度参考 | 用途 |
|---------|---------|------|
| 小标签块（2行字） | 50-54 | 模块标签 |
| 中等块（3-4行字） | 88-100 | 平台说明/数据字段 |
| 大块（5-6行字） | 130-170 | Stage输出/多内容区 |
| 超大块（7+行字） | 200+ | 多行文字区 |

## 三、字号梯度

| 用途 | CSS类 | 字号 | 颜色 |
|------|-------|------|------|
| 图表主标题 | `.title` | 20-22px 加粗 | #111827 |
| 副标题/日期 | `.subtitle` | 11px | #6b7280 |
| 大章标题 | `.section` | 12-13px 加粗 | 主题色 |
| Step标题 | `.step-head` | 11px 加粗 | 主题色 |
| 区块标题 | `.head` | 10px 600 | 主题色 |
| 正文内容 | `.body` | 9px | #4b5563 |
| 注释/标签 | `.small` | 8px | #9ca3af |
| 徽章/小标签 | `.badge` | 9px 600 | 主题色 |

**行高逻辑：**
- `.body` 行距 = 14px（y间隔）
- `.small` 行距 = 14px（y间隔）
- 避免小于12px的行距，字会挤在一起

## 五、区块结构规范

### 标准区块模板
```
rect: width×height, rx=6/8, fill=背景色, stroke=边框色, stroke-width=1.5
  rect(标题栏): width×22/24, rx=6/8, fill=标题背景色
  text(标题): text-anchor="middle", class="head/badge"
  text(内容): x=左边界+14, class="body", y=每行+14
```

### 标题栏规范
- 标题栏高度：22-24px
- 内容区 padding-top：16-18px（从标题栏底部算起）
- 正文从标题栏底部+20px开始

### 区块内边距
```
左边界 = rect.x + 14（不是10）
内容宽度 = rect.width - 28（左右各留14）
```

## 六、箭头规范

### 数据流箭头（实线）
- `stroke-width="2"`
- `marker-end="url(#id)"`
- 颜色 = 数据类型主题色

### 归档路径箭头（虚线）
- `stroke-dasharray="6,4"`
- `stroke-width="2"`
- 用于 Wiki写入/知识沉淀路径

### 反馈循环箭头
- 底部横向虚线 → 左侧向上 → 回到MEMORY

## 七、制作检查清单

在保存SVG前，逐项核查：

- [ ] 标题文字没有超出viewBox边界（x ± 20px以内安全）
- [ ] 每个区块内的文字行数 × 14px ≤ 区块height - 标题栏24px - 底部padding 14px
- [ ] 颜色使用正确（采集红/输出绿/分析紫/触发橙）
- [ ] 图例完整（所有颜色都有说明）
- [ ] 右下角有版本标注（v2.x 图名 · 日期 · 作者）
- [ ] 区块之间有足够间距（gap ≥ 10px）

## 八、常见问题处理

### 文字溢出框
**原因**：行数算少了 / 区块height估少了  
**解决**：
1. 扩大区块height（+10~20px）
2. 精简文字（合并相近内容）
3. 字号从9px降到8px（不推荐，会看不清）

### PNG截图不全
**原因**：viewport设置和viewBox不匹配  
**解决**：
```
viewport.width = viewBox.width + 40
viewport.height = viewBox.height + 40
```

### 文字被截断
**原因**：x坐标+文字宽度超出了rect边界  
**解决**：
- 对于 `text-anchor="middle"` 的文字：检查 `x + (内容宽度的一半)` 是否超出
- 用 `text-anchor="start"` + 左侧留白 14px 是最安全的
