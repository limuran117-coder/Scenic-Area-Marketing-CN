## 常见问题

**Q: 为什么有些目录不进 Git？**
A: `.openclaw/`、`.clawhub/` 是运行时状态，每次启动都重新生成，进 Git 会造成冲突和噪音。

**Q: `raw/` 和 `wiki/` 的区别是？**
A: `raw/` 是原始数据备份（如 Excel 数据导出），`wiki/` 是处理过的结构化知识。数据以 `wiki/` 为准。

**Q: `memory/` 和 `wiki/` 的区别是？**
A: `memory/` 是 Agent 的会话日志（过程），`wiki/` 是整理后的知识（结果）。Wiki 知识需要人工或半人工审核后入库。
